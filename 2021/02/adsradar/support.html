  • The ad blocking feature of AdLock is based on the use of advertising filters. A filter is a set of rules for sorting out various ads (banners, pop-ups, etc.). For high-quality blocking, we use a massive amount of advertising filters, which are based on EasyList technology. We carefully check the performance of each filter we add. If problems are found, the filters get finalized and supplemented by our experts.
    These filters specify the rules on the basis of which the application removes ads. Blocking of banners occurs differently since they are stored on third-party servers. To display a banner, the website needs to send a request to receive it from the server. AdLock checks all requests that are sent from the website page to the server. The requests that fall under the filters will be blocked, which positively affects the speed of page loading and traffic consumption.
